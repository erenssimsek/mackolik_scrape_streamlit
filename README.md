# ⚽ Maç Veri ve Oran Çekici (Streamlit)

Bu proje, seçilen tarih aralığına göre maç verilerini ve bahis oranlarını otomatik olarak çeken, görüntüleyen ve CSV / Excel olarak indirmenizi sağlayan bir Streamlit uygulamasıdır.

## 🚀 Özellikler

- 📅 Tarih aralığı seçimi (takvim üzerinden)
- ⚽ Maç bilgileri (takımlar, skorlar, tarih)
- 💰 İlgili geçmiş maçın kapanış bahis oranları (MS1, MSX, MS2, Alt/Üst 2.5)
- 📊 Tablo halinde görüntüleme
- ⬇️ CSV indirme
- 📊 Excel indirme

---

## 🛠️ Kullanılan Teknolojiler

- Python
- Streamlit
- Pandas
- Requests
- XlsxWriter
